## 💡 목표
사용자가 현재 무료로 체험하는 기능의 *제한점(Limitation)*을 명확히 인지하게 하고, 그 제한점을 해소하여 비즈니스의 완전한 가치에 접근할 수 있도록 유도합니다. (Action: 결제 페이지 이동/플랜 선택)

## 🖼️ 전체 구조 및 흐름 (Flow)
1. **헤드라인 & 문제 재확인:** 사용자 페인 포인트를 다시 언급하며 감정적 공감대 형성.
2. **핵심 가치 시각화 (The Hook):** 제한된 기능의 예시를 보여주며 아쉬움을 증폭. (Progress Bar 활용)
3. **해결책 제시 & 이점 강조:** 유료 플랜에서 얻는 구체적이고 측정 가능한 이점을 리스트업.
4. **CTA 블록:** 명확하고 긴급성을 부여하는 최종 행동 유도.

## 🧱 컴포넌트별 상세 설계 (V1.0 적용)

### 1. 섹션 헤더: [The Agitation]
*   **제목 (H2):** "잠재된 가치를 발견했습니다. 더 큰 비전을 위한 다음 단계가 필요합니다."
    *(Tone: 공감적이지만, 약간의 아쉬움을 유발하는 어조)*
*   **서브 카피:** "현재까지는 핵심 기능을 충분히 경험하셨습니다. 하지만 진정한 성장은 제한된 범위 안에서 일어나지 않습니다. AURA Pro로 무한대의 가능성을 열어보세요."
*   **디자인 요소:** 좌우 분할 레이아웃 (좌: 텍스트, 우: 시각적 단서)

### 2. 핵심 가치 시각화 섹션: [The Limitation - Progress Bar 활용]
이 부분이 가장 중요합니다. 사용자가 현재 경험하는 '불완전함'을 시각적으로 인지하게 해야 합니다.

*   **제목:** "현재 프로그레스: (사용자 이름)님의 성장 그래프"
*   **컴포넌트 1: 진행률 바 (Progress Bar - Primary):**
    *   **디자인:** 가로형 막대 그래프 형태. 현재 값(예: 65%)이 채워져 있고, **최대치(100%)를 향한 빈 공간**이 명확하게 보입니다.
    *   **상호작용 요소 (Interactive):** 'Pro 전환 시 예상 성장치'라는 레이블과 함께, 사용자가 이 섹션에 마우스를 올릴 때 (Hover) 막대가 100%로 채워지며 폭발적으로 커지는 애니메이션 효과를 추가합니다.
    *   **텍스트:** "현재 상태: 기본 기능 범위 내에서의 최적화 완료"
*   **컴포넌트 2: 제한된 기능 예시 (Card Component):**
    *   카드 3개를 나열하되, 가장 핵심적인 카드에 **'Pro Only' 배지**를 붙여 회색 처리합니다.
    *   **예시 A:** [무료] 기본 통계 분석 (데이터 개수: 100개 제한)
    *   **예시 B:** **[Pro Only] AI 기반 심층 인사이트 분석** (설명: 미처 발견하지 못한 시장의 숨겨진 니즈까지 예측합니다.)
    *   **예시 C:** [무료] 기본적인 콘텐츠 생성 (횟수 제한: 일 5회)

### 3. 이점 강조 섹션: [The Solution - Value Stack]
'결제할 이유(Why pay)'를 구체적인 리스트로 제시합니다.

*   **제목:** "AURA Pro가 제공하는 무한한 가능성 (Unlock Your Potential)"
*   **레이아웃:** 체크리스트 형태의 아이콘 기반 목록 (3~4개 항목)
*   **핵심 이점 (Benefit Cards):**
    1. **무제한 확장 (Unlimited Scale):** "분석 횟수 제한 없음. 프로젝트 규모에 관계없이 무한히 실험하세요." (아이콘: ∞)
    2. **최신 AI 모델 접근 (Premium Model Access):** "가장 빠르고 정확한 최신 LLM 엔진을 독점적으로 이용합니다." (아이콘: ✨ / Premium Tag)
    3. **맞춤형 API 연동 (Custom Integration):** "외부 시스템과의 완벽한 통합을 위한 전용 API 키를 제공받습니다." (기술적 가치 강조)

### 4. 최종 CTA 블록: [The Conversion Call]
*   **헤드라인:** "**지금, 당신의 잠재력을 완전히 해방하세요.**" (긴급성 및 자존감 어필)
*   **서브 카피:** "AURA Pro는 단순히 기능을 추가하는 것이 아닙니다. 시간과 노력을 압축하여 가장 빠른 성장을 보장하는 투자입니다."
*   **CTA 버튼:** **[💰 AURA Pro 플랜 확인하기]**
    *   *(V1.0 Button Component: High Contrast, Primary Action)*
    *   **마이크로 카피 (Micro Copy):** "3일 무료 체험 및 즉시 결제 취소 가능." (위험 부담 해소)
